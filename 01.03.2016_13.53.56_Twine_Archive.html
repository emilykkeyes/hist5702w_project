It can be used to one variety of different kinds of tabular data to help improve accuracy and quality of that data for later analysis, and can export data in many different formats. With OpenRefine it is very easy to manipulate large quantities of data without needing to have prior experience and knowledge of <font color="#C12283"><i><b>Datamancers</i></b></font> languages. OpenRefine is used on data that has already been formatted into columns or tables. ### Standford Named Entity Recognizer (NER) Stanford NER is tool that works with Java, to read through text and labels word sequences it recognizes based on the classifiers that are selected for it to look for, such as the name of a person, organization, or location. When a text file is run through NER, it produces a new file which include the tagging of the recognized entities. NER is a useful tool when working with text or data that is structured in a clearly defined way. This could mean text or data that has not been organized into columns or that does not have any clearly defined pattern repetitions.
